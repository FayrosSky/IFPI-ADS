//Compilador transforma o código fonte em um código executável de máquina(em binário). Não é só o compilador que faz esse processo: temos o compilador, o linker, montador, interpretador, ou seja, uma série de coisas que fazem parte disso que chamamos de compilador. Em resumo, o processo de compilação é a transformação do código fonte em código executável... o grande empecilho é que esse código fonte e esse executável ele só vai ser executável em uma determinada plataforma, que foi a plataforma pra qual ele foi compilado. Se tivermos outro sistemas como o linux ou o mac, eles vão ficar de fora da execução desse programa, isso porque ele foi compilado para uma prataforma específica. Se quisermos que esse sistema funcione em várias outras plataformas, vamos precisar de um mecanismo diferente. Por exemplo, se pegarmos a linguagem C, ela tem uma peculiaridade bem específica, que é ter compiladores para diversas plataformas. Assim, se pegarmos o código-fonte de linguagem C e passarmos pelos compiladores de outros computadores de outras plataformas... Esses compiladoresvão gerar códigos executáveis diferentes para cada plataforma. Pegando esses códigos executáveis e mandando para cada uma dessas plataformas, cada uma delas vai entender o programa em C. Em resumo: vai pegar o código-fonte em C, vai compilar em uma plataforma específica e essa plataforma vai gerar um código executável para essa plataforma. Isso, por muitos anos, foi visto como revolucionário. Por isso, a linguagem C tem uma importância muito grande. Por ter essa característica, a linguagem C é conhecida como linguagem multiplaforma, ou seja, uma vez utilizando o compilador certo, ele vai funcionar para múltiplas plataformas(mas em alguns casos - na maioria deles - teríamos que pegar esse código fonte em C e fazer pequenas adaptações, porque plataformas diferentes funcionam com micro instruções um pouco diferentes.então, basicamente, na prática, não funcionava muito bem como os livros falavam.). Por conta dessa característica de múltiplas plataformas, o C foi utilizado por muitos anos e até hoje ainda é para criar muitos softwares conhecidos, como por exemplo: o Firefox que é um navegador de múltiplas plataformas(Windows, Linux, Mac, dispositivos móveis). O windows é feito em C, o Linux em C, MacOS em C, IoS e o Android foram feitos em C. A evolução do C é o C++ e esse ++ diz respeito ao acréscimo da orientação a Objetos na linguagem(que é um conceito moderno.). A evolução do C++ seria mais tarde o acréscimo de ++ novamnete, formando o C#(linguagem muito moderna.)
	Em 1990, uma empresa chamada SunMycrosystems tinha um projeto: utilizando a linguagem C++ eles queriam interligar várias interfaces e fazer diversos dispositivos diferentes trabalharem entre si. Essa equipe liderada por James Gosling seria incumbida de realizar esse feito. Em 1990 isso era complicado, pois não tínhamos microprocessadores autômatos como temos hoje. A linguagem C++ não era suficiente para fazer isso e em vez de abandonarem o projeto eles criaram uma linguagem própria o Green Talk(lembrando o time de basquete chamado Green Team) para que dispositivos diferentes falassem entre si. O Green Talk foi rebatizado para OAK(carvalho) em 1991...E logo foi criado o Star Seven(*7) e em 1992 o projeto do Green Team foi engavetado pois não interessou as empresas. Até que em 1994 o Tim Berners-Lee estava criando a linguagem HTML(- linguagem interativa para web - aparecem em todos os vídeos do curso em Vídeo. Os inventos dele foram notáveis em todos os ramos da tecnologia.)... surgiu o projeto da SunMycrosystems chamado webrunner(que seria um navegador parecido com o Internet Explorer ou o Firefox). O nome Oak já estava registrado... Fizeram uma reunião e foi perguntando ao time do greeteam o que eles mais gostavam, e responderam café, mas para o café ser bom o café tinha que ser forte. Café forte era os grãos cultivado nas ilhas de JAVA... tinha uma gíria de java coffee para café forte, assim foi escolhida o nome da linguagem JAVA(por isso o símbolo é a uma xícara de café quente). Assim como foi renomeada a linguagem, o webrunner também foi renomeado se chamando agora de HotJava. A netscape aproveitando essa onda de alta da sunmycrosystems, aproveitou para chamar a linguagem que seria chamada de livescriptpara javascript como uma pequena jogadinha de marketing(mas são duas linguagens diferentes com propostas diferentes.)
