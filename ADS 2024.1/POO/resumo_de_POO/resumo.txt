Resumo de TypeScript:

Introdução:
        TypeScript é um superset do JavaScript que adiciona tipagem estática e outros recursos avançados.

Conceitos Básicos:
        Tipos de Dados:
* String
* Number
* boolean
* array
* tuple
* enum
* any
* void

Funções:
Você pode definir funções com tipos de parâmetros e de retorno:
        function soma(a: number, b: numer): number {
                return a + b;
}


Interfaces
As interfaces permitem definir a forma de um objeto:

interface Pessoa {
        nome: string;
        idade: number;

}

const pessoa: Pessoa = {
        nome: "João"
        idade: 30
};
